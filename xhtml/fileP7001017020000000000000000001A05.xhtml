Double Counting

There are three payoffs to slogging through these numbers. First, you can see clearly the double-counting problem. If we add up the value of all intermediate products (wheat, flour) as well as the final product (bread) in Figure 6.3, we get $23 ($3 + $8 + $12). The $23 number is not an accurate measure of what this simple economy produces because it includes the value of the wheat and flour multiple times. The accurate measure of GDP is the value of the only final product produced — one $12 loaf of bread.
