V is for Velocity

There is no mistake. The problem disappears if each loonie changes hands more than once in a year. If each loonie were only spent (and received) once in a year, then the limit of aggregate sales in the economy would be $1000. But if each loonie changes hands five times in a year, there could be $5000 worth of purchases and sales. The number of times a unit of money changes hands during a year is called the velocity of money. Economists use the letter V to represent the velocity of money.
